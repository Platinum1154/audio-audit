# Audio Audit

本地音频审计工具，采用浏览器界面 + Python 后端，适合快速批量审核目录里的音频文件。

## 功能

- 递归扫描任意目录下的常见音频文件
- 浏览器中试听、查看波形和频谱图
- 键盘快捷键审核：通过、移除、跳过、撤销、回退、快进快退
- 将不合格文件移动到所选根目录下的 `_audit_removed/`
- 使用 SQLite 保存审计日志，并支持导出 JSON
- 支持自定义问题标签

## 启动

```bash
uv sync --extra dev
uv run audio-audit
```

打开 `http://127.0.0.1:8000`

## 默认快捷键

- `Space`: 通过并切到下一条
- `K`: 移除并切到下一条
- `S`: 跳过并切到下一条
- `R`: 回到上一条
- `U`: 撤销上一步审核动作
- `P`: 播放 / 暂停
- `J`: 快退 5 秒
- `L`: 快进 5 秒
- `1-9`: 切换问题标签

## 说明

- 文件夹选择优先调用本机图形目录选择器；如果当前环境没有图形界面，可以手动输入绝对路径。
- 当前版本优先适配 Linux，路径和文件移动逻辑已经尽量保持跨平台。
- 波形图和频谱图通过系统 `ffmpeg` 生成；如果运行环境缺少 `ffmpeg`，可视化接口会不可用。
